# AMS Eval Feedback

This repository is for collecting feedback, bug reports, and feature requests for the **AMS Eval** iOS app.

## About AMS Eval

AMS Eval is an iOS application for monitoring Acute Mountain Sickness (AMS) using the Lake Louise Score system. It's designed for field use in remote locations, helping climbers and trekkers assess altitude sickness symptoms.

## How to Submit Feedback

### Report a Bug
If you've encountered a bug or issue with the app, please [create a bug report](https://github.com/jtdub/ams-eval-feedback/issues/new?template=bug_report.md).

### Request a Feature
Have an idea for improving AMS Eval? [Submit a feature request](https://github.com/jtdub/ams-eval-feedback/issues/new?template=feature_request.md).

### General Feedback
For general questions or feedback, [start a discussion](https://github.com/jtdub/ams-eval-feedback/discussions).

## App Features

- **Lake Louise Score Assessment** - Standardized 2018 consensus questionnaire
- **Evaluation History** - Track symptoms over time for multiple people
- **Altitude Detection** - GPS and barometric altitude tracking
- **PDF Reports** - Export evaluation history for medical records
- **Offline First** - Works without internet connectivity
- **iCloud Sync** - Optional sync across your devices
